💻 ما هو VS Code؟

Visual Studio Code

محرر نصوص مجاني ومفتوح المصدر من Microsoft، مصمم خصيصاً للمطورين

VS Code هو أشهر محرر نصوص في العالم! يستخدمه ملايين المطورين لأنه سريع، قوي، ومليء بالميزات التي تجعل البرمجة أسهل وأمتع.

حقيقة مثيرة: VS Code هو المحرر الأكثر شعبية بين المطورين حسب استطلاع Stack Overflow لعدة سنوات متتالية!

لماذا VS Code؟

  • مجاني تماماً: لا تدفع أي شيء
  • سريع وخفيف: يعمل بسلاسة حتى على الأجهزة القديمة
  • آلاف الإضافات: لكل لغة برمجة وأداة
  • Git مدمج: إدارة الكود بسهولة
  • IntelliSense: اقتراحات ذكية أثناء الكتابة
  • متعدد المنصات: Windows, Mac, Linux

⚙️ تثبيت VS Code

خطوات التثبيت:

  1. اذهب إلى code.visualstudio.com
  2. اضغط "Download for Windows/Mac/Linux"
  3. شغل ملف التثبيت
  4. اتبع التعليمات (اختر الإعدادات الافتراضية)
  5. أعد تشغيل الجهاز إذا طُلب منك

💡 نصائح التثبيت:

  • ✅ اختر "Add to PATH" أثناء التثبيت
  • ✅ اختر "Register Code as an editor for supported file types"
  • ✅ اختر "Add 'Open with Code' action to Windows Explorer"

🚀 ميزات متقدمة

1. IntelliSense - الاقتراحات الذكية

VS Code يقترح عليك الكود أثناء الكتابة:

  • أسماء المتغيرات والدوال
  • خصائص CSS
  • تاجات HTML
  • مكتبات JavaScript

2. Multi-cursor - التحديد المتعدد

حدد عدة أماكن واكتب فيها جميعاً:

  • Alt + Click: إضافة cursor جديد
  • Ctrl + D: تحديد الكلمة التالية المشابهة
  • Ctrl + Shift + L: تحديد جميع الكلمات المشابهة

3. Emmet - كتابة HTML/CSS سريعة

أمثلة Emmet:

Emmet Shortcuts 

div.container>h1+p*3


<div class="container">
    <h1></h1>
    <p></p>
    <p></p>
    <p></p>
</div>


ul>li.item$*5


<ul>
    <li class="item1"></li>
    <li class="item2"></li>
    <li class="item3"></li>
    <li class="item4"></li>
    <li class="item5"></li>
</ul>

4. Git Integration - دمج Git

VS Code يدعم Git بشكل كامل:

  • رؤية التغييرات في الملفات
  • عمل commit مباشرة
  • إدارة الفروع (branches)
  • حل التعارضات (conflicts)

⚡ تحسين سير العمل

🎯 سير عمل مطور محترف

كيف تستخدم VS Code بكفاءة عالية

1. إعداد المشروع:

  1. افتح مجلد المشروع: File > Open Folder
  2. ثبت الإضافات المطلوبة
  3. اضبط إعدادات المشروع في .vscode/settings.json
  4. أنشئ ملف .gitignore

2. الكتابة الفعالة:

  • استخدم Emmet لكتابة HTML/CSS سريعة
  • استفد من IntelliSense والاقتراحات
  • استخدم Snippets للكود المتكرر
  • استخدم Multi-cursor للتعديل المتعدد

3. التنظيم والبحث:

  • استخدم Command Palette للوصول السريع
  • استخدم Quick Open لفتح الملفات سريعاً
  • نظم الملفات في مجلدات منطقية
  • استخدم البحث المتقدم في المشروع

4. التصحيح والاختبار:

  • استخدم Live Server لمعاينة المواقع
  • استخدم Debugger المدمج
  • استفد من Terminal المدمج
  • استخدم Extensions للغات المختلفة

⚙️ إعدادات مفيدة

إعدادات أساسية مهمة:

settings.json 
{
    // حفظ تلقائي
    "files.autoSave": "afterDelay",
    "files.autoSaveDelay": 1000,
    
    // تنسيق تلقائي عند الحفظ
    "editor.formatOnSave": true,
    
    // إظهار المسافات والتابات
    "editor.renderWhitespace": "selection",
    
    // حجم التاب
    "editor.tabSize": 2,
    "editor.insertSpaces": true,
    
    // إظهار أرقام الأسطر
    "editor.lineNumbers": "on",
    
    // تفعيل Emmet لجميع الملفات
    "emmet.includeLanguages": {
        "javascript": "javascriptreact"
    },
    
    // إعدادات Terminal
    "terminal.integrated.fontSize": 14,
    
    // إعدادات الخط
    "editor.fontFamily": "Fira Code, Consolas, monospace",
    "editor.fontLigatures": true,
    "editor.fontSize": 14,
    
    // إعدادات الثيم
    "workbench.colorTheme": "One Dark Pro",
    "workbench.iconTheme": "material-icon-theme"
}
