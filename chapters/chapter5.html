🎨 ما هو CSS؟

CSS

Cascading Style Sheets
أوراق الأنماط المتتالية - لغة تنسيق تُستخدم لتصميم وتنسيق صفحات HTML

إذا كان HTML هو الهيكل العظمي للموقع، فإن CSS هو الجلد والملابس والمكياج! CSS يحول صفحة HTML العادية إلى تحفة فنية جميلة ومنسقة.

تشبيه مفيد: تخيل أن HTML هو بيت من الطوب الأحمر العادي، و CSS هو الطلاء والديكور والأثاث الذي يجعل البيت جميلاً ومريحاً!

لماذا نحتاج CSS؟

  • الجمال: تحويل الصفحات من مملة إلى جذابة
  • التنظيم: ترتيب العناصر بشكل منطقي
  • التجاوب: جعل الموقع يعمل على جميع الأجهزة
  • التفاعل: إضافة حركات وتأثيرات
  • الاحترافية: إعطاء انطباع مهني

🔗 طرق ربط CSS بـ HTML

1. CSS داخلي (Internal)

الكود:

HTML مع CSS داخلي 
<head>
    <style>
        h1 {
            color: blue;
            text-align: center;
        }
        p {
            font-size: 18px;
            color: green;
        }
    </style>
</head>

2. CSS خارجي (External)

ملف HTML:

ربط ملف CSS خارجي 
<head>
    <link rel="stylesheet" href="style.css">
</head>

ملف style.css:

style.css 
h1 {
    color: blue;
    text-align: center;
}

p {
    font-size: 18px;
    color: green;
}

3. CSS مضمن (Inline)

الكود:

CSS مضمن 
<h1 style="color: blue; text-align: center;">العنوان</h1>
<p style="font-size: 18px; color: green;">الفقرة</p>
أفضل ممارسة: استخدم CSS الخارجي دائماً! إنه أكثر تنظيماً وسهولة في الصيانة.

🎯 المحددات (Selectors)

المحددات تخبر CSS أي عناصر HTML تريد تنسيقها:

1. محدد العنصر (Element Selector)

h1 {
    color: red;
}

p {
    font-size: 16px;
}

يؤثر على: جميع عناصر h1 و p في الصفحة

2. محدد الفئة (Class Selector)

.highlight {
    background-color: yellow;
}

.big-text {
    font-size: 24px;
}

HTML: <p class="highlight">نص مميز</p>

3. محدد المعرف (ID Selector)

#header {
    background-color: blue;
    color: white;
}

#footer {
    text-align: center;
}

HTML: <div id="header">الرأس</div>

4. المحددات المتقدمة

المحدد المثال الوصف
المحدد الشامل * يؤثر على جميع العناصر
محدد الأطفال div p جميع p داخل div
محدد الطفل المباشر div > p p المباشر داخل div
محدد الأشقاء h1 + p p التي تأتي بعد h1 مباشرة
محدد الخصائص [type="text"] عناصر لها خاصية معينة

🌈 الألوان في CSS

طرق تحديد الألوان:

1. أسماء الألوان

h1 {
    color: red;
    background-color: blue;
}
red blue green yellow

2. الألوان السادسة عشرية (Hex)

h1 {
    color: #FF0000;        /* أحمر */
    background-color: #00FF00; /* أخضر */
}
#FF0000 #00FF00 #0000FF #FFFF00

3. RGB و RGBA

h1 {
    color: rgb(255, 0, 0);           /* أحمر */
    background-color: rgba(0, 255, 0, 0.5); /* أخضر شفاف */
}

RGB: Red, Green, Blue (0-255)

RGBA: مع إضافة الشفافية Alpha (0-1)

📦 نموذج الصندوق (Box Model)

كل عنصر في HTML هو عبارة عن صندوق له أربعة أجزاء:

Margin (الهامش الخارجي)
Border (الحدود)
Padding (الهامش الداخلي)
Content (المحتوى)

خصائص نموذج الصندوق:

Box Model Properties 
.box {
    /* المحتوى */
    width: 200px;
    height: 100px;
    
    /* الهامش الداخلي */
    padding: 20px;
    
    /* الحدود */
    border: 2px solid #333;
    
    /* الهامش الخارجي */
    margin: 10px;
    
    /* لون الخلفية */
    background-color: lightblue;
}
مهم: العرض الإجمالي = width + padding + border + margin

🔄 Flexbox - التخطيط المرن

Flexbox يجعل ترتيب العناصر سهلاً ومرناً:

الأساسيات:

Flexbox Container 
.flex-container {
    display: flex;
    justify-content: center;    /* محاذاة أفقية */
    align-items: center;        /* محاذاة عمودية */
    gap: 20px;                  /* المسافة بين العناصر */
}

مثال عملي:

الكود:

.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem;
    background-color: #333;
    color: white;
}

خصائص Flexbox المهمة:

الخاصية القيم الوصف
justify-content flex-start, center, space-between المحاذاة الأفقية
align-items flex-start, center, stretch المحاذاة العمودية
flex-direction row, column اتجاه العناصر
flex-wrap nowrap, wrap التفاف العناصر

📱 التصميم المتجاوب (Responsive Design)

جعل موقعك يبدو رائعاً على جميع الأجهزة:

Media Queries:

Responsive CSS 
/* للشاشات الكبيرة */
.container {
    max-width: 1200px;
    margin: 0 auto;
}

/* للأجهزة اللوحية */
@media (max-width: 768px) {
    .container {
        padding: 0 20px;
    }
    
    .grid {
        grid-template-columns: 1fr;
    }
}

/* للهواتف */
@media (max-width: 480px) {
    .text {
        font-size: 14px;
    }
    
    .button {
        width: 100%;
    }
}
جرب تصغير نافذة المتصفح! هذا الصندوق سيغير لونه حسب حجم الشاشة.

⭐ أفضل الممارسات في CSS

1. تنظيم الكود

  • استخدم تعليقات واضحة
  • نظم CSS حسب الأقسام
  • استخدم أسماء فئات وصفية
  • تجنب الـ !important
CSS منظم 
/* ========== إعادة تعيين الأنماط ========== */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* ========== المتغيرات ========== */
:root {
    --primary-color: #667eea;
    --font-size-large: 2rem;
}

/* ========== الأنماط العامة ========== */
body {
    font-family: 'Cairo', sans-serif;
    line-height: 1.6;
    color: var(--text-color);
}

/* ========== الرأس ========== */
.header {
    background-color: var(--primary-color);
    padding: 1rem 0;
}

/* ========== المحتوى الرئيسي ========== */
.main-content {
    max-width: 1200px;
    margin: 0 auto;
    padding: 2rem;
}

/* ========== التذييل ========== */
.footer {
    background-color: #333;
    color: white;
    text-align: center;
    padding: 2rem 0;
}

2. تسمية الفئات (BEM Methodology)

BEM Naming 
/* Block */
.card { }

/* Element */
.card__header { }
.card__body { }
.card__footer { }

/* Modifier */
.card--large { }
.card--featured { }
.card__header--blue { }

3. الأداء والتحسين

  • تجنب المحددات المعقدة: div > ul > li > a
  • استخدم فئات بدلاً من IDs: للتنسيق
  • قلل من استخدام @import: يبطئ التحميل
  • ضغط ملفات CSS: في الإنتاج
