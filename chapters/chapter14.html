🌶️ ما هو Flask؟

Flask

إطار عمل (Framework) بسيط ومرن لبناء تطبيقات الويب بـ Python

Flask هو أحد أشهر أطر العمل في Python! يُطلق عليه "micro-framework" لأنه بسيط وخفيف، لكن قوي بما يكفي لبناء تطبيقات معقدة. مثل السكين السويسري للمطورين!

لماذا Flask؟ لأنه بسيط للمبتدئين، قوي للمحترفين، ومرن لأي نوع من التطبيقات!

مميزات Flask:

  • بساطة: سهل التعلم والاستخدام
  • مرونة: يمكن تخصيصه لأي احتياج
  • Python: يستخدم لغة Python المحبوبة
  • مجتمع كبير: دعم ومكتبات كثيرة
  • سرعة التطوير: بناء تطبيقات سريعاً

Flask vs Django:

🌶️ Flask

  • بسيط ومرن
  • مناسب للمشاريع الصغيرة والمتوسطة
  • تحكم كامل في التطبيق
  • سهل التعلم

🎸 Django

  • شامل ومتكامل
  • مناسب للمشاريع الكبيرة
  • ميزات جاهزة كثيرة
  • أكثر تعقيداً

⚙️ تثبيت Flask

خطوات التثبيت:

# 1. تأكد من تثبيت Python python --version # 2. إنشاء مجلد المشروع mkdir my-flask-app cd my-flask-app # 3. إنشاء بيئة افتراضية (اختياري لكن مُنصح به) python -m venv flask-env # 4. تفعيل البيئة الافتراضية # Windows: flask-env\Scripts\activate # Mac/Linux: source flask-env/bin/activate # 5. تثبيت Flask pip install Flask # 6. التحقق من التثبيت python -c "import flask; print(flask.__version__)"

🛣️ Routes في Flask

Route

مسار URL يربط عنوان معين بدالة Python محددة

🗺️ أمثلة على Routes

GET / → الصفحة الرئيسية
GET /about → صفحة عن الموقع
GET /user/<name> → صفحة المستخدم
POST /contact → إرسال رسالة

أمثلة متقدمة على Routes:

from flask import Flask app = Flask(__name__) # Route بسيط @app.route('/') def home(): return 'الصفحة الرئيسية' # Route مع متغير @app.route('/user/<name>') def user_profile(name): return f'مرحباً {name}!' # Route مع نوع محدد @app.route('/post/<int:post_id>') def show_post(post_id): return f'المنشور رقم: {post_id}' # Route مع عدة HTTP methods @app.route('/contact', methods=['GET', 'POST']) def contact(): if request.method == 'POST': return 'تم إرسال الرسالة!' return 'صفحة التواصل' # Route مع query parameters @app.route('/search') def search(): query = request.args.get('q', '') return f'البحث عن: {query}'

📄 Templates في Flask

Templates تسمح لك بفصل HTML عن Python وإنشاء صفحات ديناميكية:

📁 templates/base.html
<!DOCTYPE html> <html lang="ar" dir="rtl"> <head> <meta charset="UTF-8"> <title>{% block title %}موقعي{% endblock %}</title> <link rel="stylesheet" href="{{ url_for('static', filename='css/style.css') }}"> </head> <body> <nav> <a href="/">الرئيسية</a> <a href="/about">عنا</a> </nav> <main> {% block content %}{% endblock %} </main> </body> </html>
📁 templates/home.html
{% extends "base.html" %} {% block title %}الرئيسية - موقعي{% endblock %} {% block content %} <h1>مرحباً {{ name }}!</h1> <p>اليوم هو {{ date }}</p> <h2>المنتجات:</h2> <ul> {% for product in products %} <li>{{ product.name }} - {{ product.price }} جنيه مصري</li> {% endfor %} </ul> {% endblock %}

استخدام Templates في Python:

from flask import Flask, render_template from datetime import datetime app = Flask(__name__) @app.route('/') def home(): products = [ {'name': 'لابتوب', 'price': 2500}, {'name': 'هاتف', 'price': 1200}, {'name': 'تابلت', 'price': 800} ] return render_template('home.html', name='أحمد', date=datetime.now().strftime('%Y-%m-%d'), products=products) @app.route('/user/<name>') def user_profile(name): return render_template('profile.html', username=name)

📝 التعامل مع Forms

مثال: نموذج تسجيل مستخدم جديد

📁 templates/register.html
{% extends "base.html" %} {% block content %} <h1>تسجيل مستخدم جديد</h1> <form method="POST"> <div> <label for="name">الاسم:</label> <input type="text" id="name" name="name" required> </div> <div> <label for="email">البريد الإلكتروني:</label> <input type="email" id="email" name="email" required> </div> <div> <label for="password">كلمة المرور:</label> <input type="password" id="password" name="password" required> </div> <button type="submit">تسجيل</button> </form> {% if message %} <p class="message">{{ message }}</p> {% endif %} {% endblock %}
🐍 app.py - معالجة Form
from flask import Flask, render_template, request, redirect, url_for @app.route('/register', methods=['GET', 'POST']) def register(): if request.method == 'POST': # جلب البيانات من النموذج name = request.form['name'] email = request.form['email'] password = request.form['password'] # التحقق من البيانات if len(name) < 2: return render_template('register.html', message='الاسم قصير جداً') # حفظ المستخدم (في قاعدة البيانات) save_user(name, email, password) return redirect(url_for('success')) return render_template('register.html') @app.route('/success') def success(): return '<h1>تم التسجيل بنجاح!</h1>'

🗄️ ربط Flask بقاعدة البيانات

🔗 Flask + SQLite

سنستخدم SQLite لأنها بسيطة ولا تحتاج إعداد معقد

تثبيت المكتبات المطلوبة:

pip install Flask Flask-SQLAlchemy

إعداد قاعدة البيانات:

from flask import Flask, render_template, request, redirect, url_for from flask_sqlalchemy import SQLAlchemy from datetime import datetime app = Flask(__name__) app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///users.db' app.config['SECRET_KEY'] = 'your-secret-key' db = SQLAlchemy(app) # نموذج المستخدم class User(db.Model): id = db.Column(db.Integer, primary_key=True) name = db.Column(db.String(100), nullable=False) email = db.Column(db.String(100), unique=True, nullable=False) created_at = db.Column(db.DateTime, default=datetime.utcnow) def __repr__(self): return f'<User {self.name}>' # إنشاء الجداول with app.app_context(): db.create_all() @app.route('/register', methods=['GET', 'POST']) def register(): if request.method == 'POST': name = request.form['name'] email = request.form['email'] # التحقق من وجود المستخدم existing_user = User.query.filter_by(email=email).first() if existing_user: return render_template('register.html', message='البريد الإلكتروني مُستخدم بالفعل') # إنشاء مستخدم جديد new_user = User(name=name, email=email) db.session.add(new_user) db.session.commit() return redirect(url_for('users')) return render_template('register.html') @app.route('/users') def users(): all_users = User.query.all() return render_template('users.html', users=all_users)
