🌐 ما هي استضافة المواقع؟

استضافة المواقع (Web Hosting)

خدمة توفر مساحة على خادم متصل بالإنترنت لعرض موقعك للعالم

تخيل أن موقعك هو منزل، والاستضافة هي قطعة الأرض التي ستبني عليها المنزل. بدون الأرض (الاستضافة)، لن يكون لديك مكان لبناء منزلك (موقعك)!

مثال بسيط: عندما تكتب google.com في المتصفح، يذهب طلبك إلى خوادم Google التي تستضيف الموقع!

مكونات نشر الموقع:

🏠 الاستضافة (Hosting)

  • مساحة لحفظ ملفات الموقع
  • خادم يعمل 24/7
  • اتصال سريع بالإنترنت
  • نسخ احتياطية

🌍 النطاق (Domain)

  • عنوان الموقع (مثل google.com)
  • سهل التذكر للمستخدمين
  • يشير إلى خادم الاستضافة
  • يُجدد سنوياً

🚀 خطوات نشر موقع على Netlify

1إعداد الملفات

تأكد أن ملفات موقعك جاهزة ومنظمة:

my-website/
├── index.html
├── style.css
├── script.js
├── images/
│   ├── logo.png
│   └── hero.jpg
└── pages/
    ├── about.html
    └── contact.html

2إنشاء حساب Netlify

  • اذهب إلى netlify.com
  • اضغط "Sign up"
  • سجل بـ GitHub أو البريد الإلكتروني
  • أكد حسابك

3رفع الموقع

طريقتان لرفع موقعك:

الطريقة الأولى: السحب والإفلات

  • اضغط "Deploy manually"
  • اسحب مجلد موقعك إلى المنطقة المحددة
  • انتظر انتهاء الرفع
  • احصل على رابط موقعك!

الطريقة الثانية: ربط GitHub

  • ارفع موقعك على GitHub
  • اضغط "New site from Git"
  • اختر GitHub واربط حسابك
  • اختر Repository موقعك
  • سيتم النشر تلقائياً عند كل تحديث!

4تخصيص النطاق

  • اذهب لإعدادات الموقع
  • اضغط "Change site name"
  • اختر اسماً مناسباً
  • موقعك سيصبح: yourname.netlify.app

  6. انتظر حتى 24 ساعة للتفعيل

🔒 SSL وأمان الموقع

🛡️ ما هو SSL؟

SSL (Secure Sockets Layer) هو بروتوكول أمان يشفر البيانات بين المتصفح والخادم. يظهر كقفل أخضر في شريط العنوان ويجعل رابط موقعك يبدأ بـ https://

لماذا SSL مهم؟

  • الأمان: حماية بيانات المستخدمين
  • الثقة: المستخدمون يثقون بالمواقع الآمنة
  • SEO: Google يفضل المواقع الآمنة
  • المتطلبات: بعض الميزات تتطلب HTTPS

الحصول على SSL مجاني:

  • Netlify يوفر SSL مجاني تلقائياً
  • Let's Encrypt يوفر شهادات مجانية
  • Cloudflare يوفر SSL مجاني
  • معظم مقدمي الاستضافة يوفرونه مجاناً

🧠 اختبر فهمك

1. ما هو الفرق بين الاستضافة والنطاق؟

  • لا يوجد فرق
  • الاستضافة مساحة، النطاق عنوان
  • النطاق أهم من الاستضافة
  • الاستضافة مجانية دائماً

2. أي منصة مناسبة للمواقع الثابتة؟

  • قاعدة بيانات MySQL
  • Netlify
  • WordPress فقط
  • لا يمكن نشر المواقع الثابتة

3. ما هو SSL؟

  • نوع من قواعد البيانات
  • بروتوكول أمان للمواقع
  • لغة برمجة
  • نوع من الخوادم

4. لماذا نحتاج CDN؟

  • لكتابة الكود
  • لتسريع تحميل الموقع
  • لتصميم الموقع
  • لكتابة HTML
