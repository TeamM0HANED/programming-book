🌐 ما هو HTML؟

HTML

HyperText Markup Language
لغة ترميز النصوص التشعبية - اللغة الأساسية لبناء صفحات الويب

HTML ليس لغة برمجة، بل لغة ترميز. تخيل أنك تكتب مقالاً في برنامج Word وتريد تنسيقه: تجعل العنوان كبيراً وغامقاً، والفقرات عادية، وتضيف قوائم وصور. HTML يفعل نفس الشيء لصفحات الويب!

💡 مثال بسيط

عندما تكتب في HTML:

<h1>مرحباً بالعالم!</h1>
<p>هذه أول صفحة ويب لي.</p>

المتصفح يعرضها هكذا:

مرحباً بالعالم!

هذه أول صفحة ويب لي.

🏗️ هيكل صفحة HTML

كل صفحة HTML لها هيكل أساسي ثابت:

<!DOCTYPE html> <!-- نوع المستند --> <html lang="ar" dir="rtl"> <!-- بداية الصفحة --> <head> <!-- معلومات الصفحة --> <meta charset="UTF-8"> <!-- ترميز النص --> <title>عنوان الصفحة</title> <!-- عنوان التبويب --> </head> <body> <!-- محتوى الصفحة --> <h1>العنوان الرئيسي</h1> <p>فقرة من النص.</p> </body> </html> <!-- نهاية الصفحة -->

شرح الأجزاء:

  • <!DOCTYPE html> - يخبر المتصفح أن هذا مستند HTML5
  • <html> - العنصر الجذر لكل صفحة
  • <head> - معلومات عن الصفحة (لا تظهر للمستخدم)
  • <body> - المحتوى المرئي للصفحة

الفقرات والنصوص

الكود:

<p>هذه فقرة عادية من النص.</p>
<p>هذا نص <strong>غامق</strong> و <em>مائل</em>.</p>
<p>يمكنك <u>تسطير النص</u> أيضاً.</p>

القوائم

قائمة مرقمة:

<ol>
    <li>العنصر الأول</li>
    <li>العنصر الثاني</li>
    <li>العنصر الثالث</li>
</ol>
  1. العنصر الأول
  2. العنصر الثاني
  3. العنصر الثالث

قائمة نقطية:

<ul>
    <li>تفاح</li>
    <li>موز</li>
    <li>برتقال</li>
</ul>
  • تفاح
  • موز
  • برتقال

🔗 الروابط والصور

الروابط (Links)

الكود:

<a href="https://www.google.com">اذهب إلى جوجل</a>
<a href="page2.html">صفحة أخرى في موقعي</a>
<a href="mailto:someone@example.com">أرسل إيميل</a>
اذهب إلى جوجل
صفحة أخرى في موقعي
أرسل إيميل

الصور (Images)

الكود:

<img src="image.jpg" alt="وصف الصورة" width="200">

ملاحظة: خاصية alt مهمة جداً لوصف الصورة للمكفوفين ومحركات البحث.

⚙️ الخصائص (Attributes)

الخصائص تعطي معلومات إضافية للعناصر:

الخاصية الاستخدام مثال
id معرف فريد للعنصر <div id="header">
class فئة للتنسيق <p class="intro">
style تنسيق مباشر <p style="color: red;">
title نص يظهر عند التمرير <p title="معلومة إضافية">

🚀 مشروع عملي: صفحة شخصية

الآن سنبني صفحة شخصية بسيطة تحتوي على:

  • عنوان رئيسي باسمك
  • فقرة تعريفية عنك
  • قائمة بهواياتك
  • روابط لحساباتك على مواقع التواصل
personal-page.html 
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>صفحتي الشخصية</title>
</head>
<body>
    <h1>أحمد محمد</h1>
    <h2>مطور ويب مبتدئ</h2>
    
    <p>مرحباً! أنا أحمد، أتعلم البرمجة وتطوير الويب. 
    أحب التكنولوجيا وأسعى لبناء مواقع ويب رائعة.</p>
    
    <h3>هواياتي:</h3>
    <ul>
        <li>البرمجة</li>
        <li>القراءة</li>
        <li>الرياضة</li>
        <li>الألعاب</li>
    </ul>
    
    <h3>تواصل معي:</h3>
    <p>
        <a href="mailto:ahmed@example.com">البريد الإلكتروني</a> | 
        <a href="https://twitter.com/ahmed">تويتر</a> | 
        <a href="https://github.com/ahmed">جيت هاب</a>
    </p>
</body>
</html>

🧠 اختبر فهمك

1. ما هو العنصر المستخدم للعنوان الرئيسي؟

  • <h1>
  • <title>
  • <header>
  • <head>

2. أي عنصر يُستخدم لإنشاء رابط؟

  • <link>
  • <a>
  • <url>
  • <href>

3. ما الفرق بين <ol> و <ul>؟

  • لا يوجد فرق
  • <ol> قائمة مرقمة، <ul> قائمة نقطية
  • <ul> قائمة مرقمة، <ol> قائمة نقطية
  • كلاهما نفس الشيء
