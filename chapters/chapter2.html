🌍 عالم لغات البرمجة

هناك المئات من لغات البرمجة في العالم، كل واحدة منها صُممت لحل مشكلات معينة أو للعمل في بيئات محددة. تماماً مثل اللغات البشرية، كل لغة برمجة لها قواعدها وخصائصها الفريدة.

هل تعلم؟ أن هناك أكثر من 700 لغة برمجة مختلفة، لكن المبرمجين يستخدمون عادة 10-20 لغة فقط في معظم المشاريع!

📊 تصنيفات لغات البرمجة

1. حسب مستوى التجريد

⚡

لغات منخفضة المستوى

قريبة من لغة الآلة، سريعة جداً لكن صعبة الكتابة

أمثلة: Assembly, Machine Code
⚖️

لغات متوسطة المستوى

توازن بين السرعة وسهولة الاستخدام

أمثلة: C, C++, Rust
😊

لغات عالية المستوى

سهلة الفهم والكتابة، قريبة من اللغة البشرية

أمثلة: Python, JavaScript, Java

2. حسب طريقة التنفيذ

🔄 لغات مُفسرة (Interpreted)

  • تُنفذ سطر بسطر
  • سهلة التطوير والاختبار
  • أبطأ في التنفيذ
  • تحتاج المفسر لتشغيل البرنامج
أمثلة: Python, JavaScript, PHP

⚡ لغات مُجمعة (Compiled)

  • تُترجم إلى كود الآلة مسبقاً
  • أسرع في التنفيذ
  • لا تحتاج مترجم عند التشغيل
  • أصعب في التطوير والاختبار
أمثلة: C, C++, Go, Rust

🏆 أشهر لغات البرمجة

🐍
Python
1991
سهل للمبتدئين

الاستخدامات: الذكاء الاصطناعي، علم البيانات، تطوير الويب، الأتمتة

لماذا مشهورة؟ سهلة التعلم، مكتبات ضخمة، مجتمع كبير

print("مرحباً بالعالم!") name = input("ما اسمك؟ ") print(f"أهلاً {name}!")
🌐
JavaScript
1995
متوسط

الاستخدامات: تطوير الويب، تطبيقات الجوال، تطبيقات سطح المكتب

لماذا مشهورة؟ لغة الويب الأساسية، تعمل في المتصفح والخادم

console.log("مرحباً بالعالم!"); let name = prompt("ما اسمك؟"); alert(`أهلاً ${name}!`);
☕
Java
1995
متوسط

الاستخدامات: تطبيقات المؤسسات، تطبيقات Android، أنظمة كبيرة

لماذا مشهورة؟ "اكتب مرة، شغل في أي مكان"، أمان عالي، أداء جيد

public class HelloWorld { public static void main(String[] args) { System.out.println("مرحباً بالعالم!"); } }
⚡
C++
1985
صعب

الاستخدامات: الألعاب، أنظمة التشغيل، البرمجيات عالية الأداء

لماذا مشهورة؟ سرعة عالية جداً، تحكم كامل في الذاكرة

#include <iostream> using namespace std; int main() { cout << "مرحباً بالعالم!" << endl; return 0; }

🤔 كيف تختار اللغة المناسبة؟

🎯 حدد هدفك

  • تطوير الويب: JavaScript, Python, PHP
  • تطبيقات الجوال: Java/Kotlin (Android), Swift (iOS)
  • الذكاء الاصطناعي: Python, R
  • الألعاب: C++, C#, JavaScript
  • تطبيقات سطح المكتب: C++, Java, C#

⏰ اعتبارات أخرى

  • الوقت المتاح: Python أسرع للتعلم من C++
  • فرص العمل: ابحث عن الطلب في منطقتك
  • المجتمع: لغات بمجتمع كبير = مساعدة أكثر
  • المستقبل: تأكد أن اللغة لها مستقبل
  • الشركات: ما تستخدمه الشركات التي تريد العمل بها
نصيحة للمبتدئين: ابدأ بـ Python أو JavaScript. هما أسهل للتعلم ولهما استخدامات واسعة وفرص عمل ممتازة!

3. ما الفرق بين اللغات المُفسرة والمُجمعة؟

  • لا يوجد فرق
  • المُفسرة تُنفذ سطر بسطر، المُجمعة تُترجم مسبقاً
  • المُجمعة أبطأ من المُفسرة
  • المُفسرة تحتاج ترجمة مسبقة
